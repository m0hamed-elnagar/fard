# Design Documentation - Fard (Qada Tracker)

## Design System

This project follows the **Android Material You (Material 3)** design system.

### Reference Figma Design
[Android Material You UI Kit - Free - Community](https://www.figma.com/design/LglvCa6Cxj53J4HKpEqw5g/Android-Material-You-UI-Kit--Free---Community-?node-id=0-1&p=f&t=IKjRNF9xiT3OdmrW-0)

### Design Philosophy
- **Material 3 (Material You):** Utilizing the latest Material Design standards, including large rounded corners (`28dp+`), distinctive color palettes, and modern typography.
- **Aesthetic:** Dark-focused with "Emerald & Gold" accents, providing a premium Islamic-inspired look.
- **Components:** Using Material 3 `ColorScheme`, `NavigationBar`, and `Card` components with updated styling.

### Key Styling Tokens (Planned/Implemented)
- **Primary Color:** Emerald Green (`#2E7D32`)
- **Accent Color:** Gold/Amber (`#FFD54F`)
- **Background:** Deep Dark (`#0D1117`)
- **Corner Radius:** `24.0` to `28.0` for containers and cards.
- **Typography:** `GoogleFonts.outfit` for a modern, clean look.
